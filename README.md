# 🐸 Wednesday Frog Sender (Pepe Edition)

Автоматизированный бот для поздравления друзей с «Лямгущачьей средой». Работает полностью автономно через **GitHub Actions** — никакого локального сервера не требуется.

## ✨ Особенности
- **Двойной стек**: В проекте используются и `requests` (для быстрой проверки статуса), и `httpx` (для асинхронного получения данных). Отличная шпаргалка!
- **Zero-Cost Hosting**: Запускается бесплатно на серверах GitHub по расписанию (Cron).
- **Multi-User**: Умеет рассылать поздравления сразу списку друзей.
- **Fail-safe**: Если API прилегло, бот отправит резервную лягушку из репозитория.

## 🛠 Технологии
- **aiogram 3.x** — современный асинхронный фреймворк для Telegram.
- **httpx** — асинхронные HTTP-запросы.
- **requests** — классические синхронные запросы.
- **GitHub Actions** — планировщик и среда запуска.

## 🚀 Как настроить под себя

1. **Клонируй репозиторий**:
```bash
git clone https://github.com
```
2. **Настрой секреты на GitHub** (Settings > Secrets and variables > Actions):
- BOT_TOKEN: Токен от @BotFather.
- FRIEND_ID: ID пользователей через запятую (например: 12345,67890).
3. **Загрузи резервное фото:**
Положи файл backup_frog.jpg в корень проекта, чтобы бот всегда имел запасной вариант.
4. **Проверь расписание:**
В файле .github/workflows/wednesday_frog.yml настроено время:
cron: '0 7 * * 3' — это 10:00 по МСК каждую среду
